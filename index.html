Numerische Berechnungen von Elektro-magnetischen Phänomenen

Auf dieser Site stelle ich vor, wie ein physikalisches Phänomen - nämlich die Lichtausbreitung in dielektrischen, runden Resonatoren mathematisch beschrieben werden kann. Dabei kommen Theorie und Code-Beispiele zum Einsatz. Im Resultat werden Lösungen von Differentialgleichungen mit einem numerischen Verfahren vorgestellt. 

import numpy as np

x  = np.pi + 23

for i in range 1:
    print("hello world")

Grundlagen der Elektromagnetischen Wellen

Elektromagnetische Wellen gehen aus der harmonischen Lösung der Maxwellgleichungen hervor. Eine Beschreibung und Einbettung in den Zusammenhang des hier vorgestellten Phänomens ist über den Link Maxwell nachvollziehbar.

Finite Differenzen Methode

Lösungen der Maxwellgleichungen können numerisch über das Finite-Differenzen Verfahren approximiert werden. Dabei werden zu untersuchende Strukturen auf einem Rechengitter projiziert. Ein einfaches Rechengitter lässt sich über folgenden Code in C realisieren: 

int Sx = 1;
int Sy = 1;
int Nx = 20;
int Ny = 20;

int x0 = 0.3;
int y0 = 0.6;
int rx = 0.2;
int ry = 0.35;

//Erstellung des Grids
int dx = Sx/Nx;
